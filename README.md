# Interactive Roadmap Tracker

A beginner-friendly React project that starts with local authentication and then opens a visual roadmap dashboard.

## Project setup

- `package.json` stores project scripts and dependencies.
- `vite.config.js` configures Vite for React.
- `index.html` is the browser entry page.
- `src/main.jsx` starts the React app.

## Planned folder structure

```text
src/
  components/
    auth/
    dashboard/
    roadmap/
  data/
  hooks/
  styles/
  utils/
```

## Run locally

```bash
npm install
npm run dev
```
